If you use the current version of our Android SDK, distributed tracing works out of the box for any HTTP request sent with OkHttp, Apollo or Apollo 3, if our integration for it is enabled.
If you use SDK version `6.24.x` or below, you need to follow the <PlatformLink to="/usage/distributed-tracing/custom-instrumentation/">Custom Instrumentation</PlatformLink> set up steps.
